# IELTS Writing Exercise

这是Lucas.F的雅思写作练习，旨在于提升自我的英文写作技术

IELTS写作分为两个部分：

- Task 1: 主要为客观叙述一个图、表、流程图
- Task 2: 主要为议论文，论述某个观点并加以解释

## Task 1 Types

1. Line Graph
2. Bar Chart
3. Pie Chart
4. Table
5. Flow Chart
